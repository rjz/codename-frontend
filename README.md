Codenames frontend
===============================================================================

[![Build Status](https://travis-ci.org/rjz/codenames-frontend.svg?branch=master)](https://travis-ci.org/rjz/codenames-frontend)

A teenie-tiny toy frontend.


Installation
-------------------------------------------------------------------------------

Install and build:

    $ npm install
    $ npm start

Testing
-------------------------------------------------------------------------------

Lint and run test suite:

    $ npm test
